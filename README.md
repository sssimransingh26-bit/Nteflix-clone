# Netflix Clone

A Netflix-inspired movie browsing website built using HTML, CSS, and JavaScript with data fetched from the TMDB API.

## Features

* Browse trending movies and TV shows
* Dynamic movie data using TMDB API
* Responsive UI inspired by Netflix
* Movie posters and ratings display
* Clean frontend built with vanilla JavaScript

## Tech Stack

* HTML5
* CSS3
* JavaScript
* TMDB API

## Project Structure

```text
project/
│
├── index.html
├── appm.js
├── stylesm.css
├── config.js
└── .gitignore
```

## Setup Instructions

### 1. Clone the repository

```bash
git clone https://sssimransingh26-bit.github.io/Nteflix-clone/
```

### 2. Open the project folder

```bash
cd netflix-clone
```

### 3. Create `config.js`

Create a file named `config.js` and add:

```javascript
const TMDB_API_KEY = "YOUR_TMDB_API_KEY";
```

### 4. Run the project

Open `index.html` in your browser or use VS Code Live Server.
